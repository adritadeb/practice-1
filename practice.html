Semantic tags

Semantic tags provide the information of the content systametically. We can arrange our code with it. Semantic tags describe the purpose of the element and the type of content that is inside them. By using these tags one can easily understand the information about the content. That's why these are called semantic tags. The examples of semantic tags are - header, footer, article, main, aside etc.
